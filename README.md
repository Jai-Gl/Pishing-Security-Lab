# 🛡️ Laboratorio de Seguridad: Simulador de Phishing

Este proyecto es una herramienta educativa de ciberseguridad diseñada para analizar y verificar vulnerabilidades ante ataques de ingeniería social, específicamente enfocada en la simulación de correos electrónicos fraudulentos (Phishing).

## 📋 Descripción del Proyecto

El objetivo de este laboratorio es demostrar cómo funcionan los ataques de suplantación de identidad para educar a los usuarios y fortalecer las medidas de seguridad. La simulación imita la interfaz de servicios conocidos para evaluar la capacidad de detección de amenazas.

## ⚠️ Aviso de Uso Ético (Disclaimer)

Este proyecto ha sido creado estrictamente con fines **educativos y de investigación**. El uso de estas herramientas para realizar actividades ilícitas es responsabilidad del usuario y está fuera del propósito de este repositorio.

## ✨ Características Principales

* **Simulación de Interfaz:** Réplica visual de formularios de inicio de sesión.
* **Análisis de Vulnerabilidades:** Identificación de puntos críticos en la interacción del usuario.
* **Enfoque Pedagógico:** Diseñado como un Proyecto Mínimo Viable (MVP) para concientización en ciberseguridad.

## 🛠️ Tecnologías y Herramientas

* **HTML5 / CSS3**: Estructura y diseño de la interfaz de simulación.
* **Lógica de Backend**: Manejo de peticiones y captura de datos (puedes especificar si usaste PHP, Python u otro).
* **Entorno de Pruebas**: Laboratorio controlado para ejecución segura.

## 📁 Contenido del Repositorio

* `Simulador de phishing.rar`: Contiene el código fuente y los activos necesarios para desplegar el laboratorio.
* `LÉAME.md`: Documentación del proyecto.

## 🚀 Cómo usar el laboratorio

1. Descarga el archivo comprimido `.rar`.
2. Descomprime los archivos en un servidor local (como XAMPP o WAMP) o un entorno de pruebas controlado.
3. Configura los parámetros de red necesarios.
4. Ejecuta el archivo principal para iniciar la simulación.
